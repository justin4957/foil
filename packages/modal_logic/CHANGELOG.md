# Changelog

All notable changes to modal_logic will be documented in this file.

## [0.1.0] - 2026-01-14

### Added

#### Phase 1 - Foundation  
- 7 modal system profiles (PR #108)
- Enhanced error reporting with 6 categories and 18 error codes (PR #109)
- Formula pattern library with 25 reusable patterns (PR #110)
- CI/CD pipeline with 3 GitHub Actions workflows (PR #119)

#### Phase 2 - Strategic Enhancements
- 12 dataset-specific templates for FOLIO, LogiQA, InPhO (PR #120)
- Batch verification across multiple systems (PR #121)
- System comparison with consensus analysis (PR #121)
- 4-tier API authentication and rate limiting (PR #122)

#### Phase 3 - User Experience
- 7 complexity metrics with optimization suggestions (PR #123)
- Interactive Monaco editor with syntax highlighting (PR #124)

#### Core Capabilities
- Natural language to formal logic translation via LLM
- Z3-powered formal verification
- Countermodel generation for invalid arguments
- Repair suggestions
- Multi-format visualization (Mermaid, Graphviz, LaTeX, Markdown)
- REST API, WebSocket, CLI, Web UI interfaces
- PostgreSQL persistence
- Comprehensive testing (555+ tests)
