# Changelog

All notable changes to z3_gleam will be documented in this file.

## [0.1.0] - 2026-01-14

### Added

- SMT expression construction (Boolean, Integer, Real, Array types)
- Modal logic support with Kripke semantics
- Standard translation to first-order logic
- Countermodel extraction and evaluation
- Solver interface (push/pop, assertions, check-sat)
- SMT-LIB format compilation
- Model evaluation utilities
- Configuration and tactics support
- Timeout handling with BEAM integration
- Performance benchmarking utilities
- Support for 7 modal systems (K, T, K4, S4, S5, KD, KD45)
- Comprehensive test suite
